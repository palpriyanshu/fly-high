# Fly High 

This app shows real-time flight status.

## Access website on

https://fly-high-priyanshus-projects-8099d868.vercel.app/

### Local Set up

#### Installation 
    > npm install

#### Run Locally
    > npm start

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

#### Run Test
    > npm run test

#### Run Build
    > npm run build
